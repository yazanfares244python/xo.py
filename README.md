# Tic-Tac-Toe AI Game Project

## Description
This is a Python-based implementation of the classic Tic-Tac-Toe game. The project features an AI opponent that you can challenge, making it both a fun and engaging way to play this timeless game. 

## Installation
To get started with the Tic-Tac-Toe AI game, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yazanfares244python/xo.py.git
   ```
2. Navigate to the project directory:
   ```bash
   cd xo.py
   ```
3. Make sure you have Python installed. You can download it from [python.org](https://www.python.org/downloads/).

## Usage
To run the game, simply execute the following command in your terminal:
```bash
python main.py
```
Now you can start playing!

## Features
- Play against a smart AI that makes optimal moves.
- Simple and clear interface for easy gameplay.
- Ability to play multiple rounds.
- Keeps track of scores.

## How to Play
1. The game is played on a 3x3 grid.
2. Players take turns to place their marks (X or O) in an empty cell.
3. The first player to align three of their marks vertically, horizontally, or diagonally wins.
4. If all cells are filled without any player winning, the game ends in a draw.

Enjoy playing Tic-Tac-Toe against the AI!
